## Introduction

This is the Top-Down Dungeon Crawler game made with [libgdx](https://libgdx.com/).

## Features

- exquisite maps
- high score board
- in-game shop
- pay2win system


## Tech Stack

- libGDX
- Gradle
- Java
- Postgres
- React
- JavaScript
- Typescript

### Prerequisites

- Java install
- Internet connection (for highscore upload)
